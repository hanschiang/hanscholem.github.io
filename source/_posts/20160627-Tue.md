## 在 Macbook Air 安裝 Django

- OS: OS X El Captain 10.11.5
- virtualenv, python 2.7.11, django 1.9.7

```
$ pip install virtualenv

$ mkdir project
$ cd project

$ virtualenv ENV
    # 在 project 資料夾裡建立虛擬環境

$ pip install django
```
